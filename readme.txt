まぜまぜ画像

製作者：Matsu-Kiham
バージョン：1.00

画像を加工してGIFアニメを作成するWindowsデスクトップアプリです．

注意：作成されるGIFアニメは人によっては驚くこともあるかもしれない
ので，驚きやすい人は注意してください．

導入方法：
mazemaze.exeをダウンロードして実行してください．
また，Form1.vbとForm1.Designer.vbはmazemaze.exeのソースコードです．

アプリの操作法：
mazemaze.exeを実行すると「まぜまぜ」，「ばらばら」，「とびとび」，
「ぐるぐる」とそれぞれ書かれた４つのボタンが表示されます．
それぞれのボタンをクリックすると，画像ファイルの選択画面が開くので，
GIFアニメの元になる画像ファイルを選択して開くボタンをクリックして
ください．画像ファイルは.pngと.jpgに対応しています．
続いて，作成したGIFアニメを保存するフォルダの選択画面が開くので，
GIFアニメを保存したいフォルダを選択してOKボタンをクリックしてくだ
さい．
GIFアニメが作成されるまで，数秒から数分ほど時間がかかります．
GIFアニメの作成が完了すると効果音が鳴り，「画像の作成に成功しました」という表示が出るのでOKボタンを押してください．
選択したフォルダに.gifの拡張子が付いたファイルが作成されていれば，
それがGIFアニメのファイルです．
ファイル名は日付（例えば，2020年7月8日 14:29:38であれば，
200708142938）の後に，「まぜまぜ」ならmaze，「ばらばら」ならbara，
「とびとび」ならtobi，「ぐるぐる」ならguruがそれぞれ続く形式になります．
作成したGIFアニメのファイルはInternet ExplorerやGoogle chrome等の
ブラウザで見ることができます．
